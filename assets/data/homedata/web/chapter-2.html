關於console Object，只要在Developer tools的Console輸入"console"就會發現console除了大家最常用到的"console.log"外，還有很多其他功能，接下來讓我們探討其中幾項用法。


style:
console所輸出的文字是可以有不同的style的，就像是html裡的物件一樣，在Facebook打開DevTools，就會看到Console的文字是有編輯過的，而用法如下(注意到"%c")：

Console
  console.log('%cwildMio', "color: blue; font-size: 50px;");
  //or
  console.log('%cwild%cMio', "color: red", "color: blue; font-size: large");

console.assert(Assertion, ErrorMessage):
如果Assertion結果為true，則甚麼都不會輸出，
如果Assertion結果為false,則會丟出錯誤訊息，例如：

javascript.js
  console.log('Assertion === true');
  console.assert(true, 'nothing happen');
  console.log('Assertion === false');
  console.assert(false, '[Error]: Something error, you can log error message here');
}

console.clear():
清空Console內容。


console.count(label):
此功能可以用來追蹤特定區域程式碼執行次數，例如以下：

javascript.js
  var label = 'Function is called';
  function mio() {
    console.count(label);
  }
  mio();
  mio();
  mio();
  }

console.dir(object) & console.dirxml(object):
讓物件以不同的型式(javscript or html)表現出來，例如：

javascript.js
  console.dir(document.body);
  console.dirxml(document.body);
  console.log(document.body);
  //console.dirxml(document.body) === console.log(document.body)

console.error():
將輸出文字的風格轉變為error風(紅紅der)。


console.group(), console.groupCollapsed() & console.groupEnd():
將多行log在Console輸出成可收合的型式，例如：

javascript.js
  console.group('User');//'User' === tag
  console.log('mio');
  console.group('Age');//group() 可替換成 groupCollapsed()，差別在於一開始是否為展開狀態
    console.log(100);
    console.log(99);
  console.groupEnd();
  console.log('leo');
  console.log('you');
  console.groupEnd();

console.info():
輸出的文字最前端匯有一個"i"圖示。(chrome無，saferi有)


console.log():
console.log的其他用法，使用"%s", "%d", "%3.3f"，例如：

javascript.js
  console.log("The string %s, The number %d, The float %4.2f. %s", "wildMio", 38, 3.14, 'New Year.');

console.table():
將array或object以table的型式表現，可以依照屬性排序，例如：

javascript.js
  let data = [{id: 0, name: 'mio', skill: 'coding'}, {id: 1, name: 'leo', skill: 'guitar'}, {id: 2, name: 'you', skill: 'all'}];
  console.table(data);

console.time(label) & console.timeEnd(label):
用來追蹤跑完指定程式碼所花的時間，例如：

javascript.js
  console.time('for loop label');
  for(let i = 0; i < 100000000; i ++) { var temp = i * 2; }
  console.timeEnd('for loop label');

console.trace():
用來追蹤function的調用非常好用!例如：

javascript.js
  function firstFunc() { secondFunc(); }
  function secondFunc() { thirdFunc(); }
  function thirdFunc() { console.trace('trace stack') } //'trace stack' === tag
  firstFunc();

console.warn():
以警告形式顯示輸出。
